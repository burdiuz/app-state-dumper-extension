Privacy Policy for App State Dumper

Last updated: September 22, 2024

Introduction

This Privacy Policy explains how App State Dumper operates regarding user data.

No Data Collection

This Chrome extension does not collect, store, or transmit any personal information, browsing data, or any other data from users. It only stores data configured and provided by user via user interface. At any point of time user is able to retrieve stored data in full and modify it, removing extension will also erase all stored data.

Changes to This Privacy Policy

This Privacy Policy may be updated from time to time. If there are changes, the updated policy will be posted on this page with the revised date.
